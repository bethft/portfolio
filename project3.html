Project Overview

About Temptunes

This project uses Flask and Spotify's API to create a web app that generates personalized playlists based on real-time weather data. Users can log in with their Spotify account, view the current weather, and enjoy a curated playlist that matches the weather's mood.

Research

Why Make This?

Our task was to create a website that integrates two APIs. During our brainstorming session, we discussed how we love matching different types of music to the weather. However, this can often be a tedious process since finding the perfect songs to make a playlist takes time.

(03)

Research

Problem

In our brainstorming session, we identified a common pain point: finding a song that reflects the weather can be time-consuming, sometimes taking up to 10 minutes to find just one song. Creating a playlist for specific weather can take even longer, and the repetition of each playlist can get tiresome, often forcing us to create new ones. This is why we created Temptunes.

(04)

Ideate

What did we do?

We decided to combine the Spotify and Weather API to create a personalized playlist for users based on their geographic location. Using Python (Flask), CSS, and HTML to create Temptunes.
